bruteforceSHA256
================

example implementation for bruteforcing SHA-256 using MPI

## Build

To build with resp. without MPI, change to MPI resp. noMPI dir and execute:

```bash
cmake .
```
```bash
make
```
#### Build Dependencies
* cmake
* make
* gcc
* gcc-c++
* openmpi-devel
* libopenssl-devel
